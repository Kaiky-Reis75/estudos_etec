# Operações Básicas – C#

## Descrição

Conjunto de atividades desenvolvidas para praticar operações básicas e cálculos matemáticos
utilizando a linguagem C# em aplicações de console.

## Objetivo

Aplicar conceitos fundamentais de programação, como:

- operações aritméticas
- entrada e saída de dados
- uso de variáveis
- lógica de cálculo

## Atividades

Cada pasta contém uma atividade independente:

1. Cálculo de média
2. Cálculo de raio de círculo
3. Cálculo de salário com INSS
4. Conversor de Fahrenheit para Celsius
5. Cálculo de salário por produção
6. Cálculo de salário por hora

## Tecnologias utilizadas

- C#

## Observações

As atividades foram desenvolvidas anteriormente durante o curso técnico na ETEC Dr. Hermílio Hernadez Aguilar
e estão sendo organizadas e publicadas posteriormente no GitHub para fins de estudo e portfólio.
